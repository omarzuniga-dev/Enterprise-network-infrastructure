# Enterprise-network-infrastructure
Network infrastructure project with Windows Server, Linux server, Active Directory, DNS, DHCP, SSH, Apache, MariaDB, and client configuration in a virtualized environment.

# Hybrid Server Environment

Infrastructure project using Windows Server and Linux.

## Technologies
- Windows Server 2016
- Active Directory
- DNS
- DHCP
- Linux Server
- Apache
- MariaDB
- SSH
- VirtualBox

## Features
- Domain configuration
- Client-server communication
- Network services setup
- Remote administration
- Web server deployment
